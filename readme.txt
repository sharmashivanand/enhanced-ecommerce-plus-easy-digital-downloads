Enhanced Ecommerce Tracking in Google Analytics for Easy Digital Downloads

== Description ==

A simple yet powerful plugin that let's you benifit from Enhanced Ecommerce Tracking in Google Analytics for Easy Digital Downloads.

This plugin has no dependencies and works independently of any other Google Analytics tracking you have on the site without any duplication.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/enhanced-ecommerce-plus-easy-digital-downloads` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the wp-admin->EDD Enhanced Ecom screen to configure the plugin

== Frequently Asked Questions ==

= The plugin is not outputting Google Analytics Code =

It's not supposed to. All the hits are fired server-side.

= Does this plugin work with Google Analytics 4 properties? =

Not yet but soon.

= Do I need to use any other Google Analytics plugin with this? =

This plugin is unrelated to any other plugins implementing Google Analytics. You may or may not use them regardless of this plugin.

= What else do I need to use this plugin? =

If you haven't already, turn on enhanced ecommerce for a view: https://support.google.com/analytics/answer/6032539
